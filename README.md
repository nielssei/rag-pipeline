A Retrieval-Augmented Generation (RAG) pipeline built on Temporal for durable workflow orchestration. Ingests PDFs, chunks and embeds them into a Qdrant vector store, then answers queries via dense retrieval, reranking, and LLM synthesis. My first project working with vector databases and workflow engines. Used it myself to build a Knowledge Base for my University documents, but later migrated my setup to Claude Code + Obsidian :D
